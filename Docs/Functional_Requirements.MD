### Functional Requirements

+ Media ingestion pipeline for images.
+ Media ingestion pipeline for audio.
+ Media ingestion pipeline for video.
+ Media ingestion pipeline for PDF.
+ Media ingestion pipeline for URL-based evidence.
+ Metadata extraction and for uploaded artifacts.
+ Tamper analysis of media and its metadata.
+ Explainable risk scoring that combines multiple detection signals.
+ Evidence case management for investigators or journalists.
+ Searchable repository of analyzed cases and findings.
+ Comparison tools for side-by-side media inspection and timeline reconstruction.
+ Report generation summarizing findings, confidence, and limitations.
